    • expectations & reminders

    This course is about Web Development. This means you should conduct yourself as Web Developers!

    What does this mean? It means you should understand the following:

  • FTP If you don't understand FTP (File Transfer Protocol) right now, that's ok. After week 2 - don't ask me. This is a required skill
  • File/Folder Names While it is technically possible, my recommendation is that you do NOT use capital letters, and I require that you do NOT use spaces in your file or folder names. Special Characters (those characters over the numbers on your keyboard) just won't work.
  • Web Space The student web space is: students.thenet.ca/(student_name) Notice that there is no www at the front
  • Upkeep Each week, you should have your course web space (we'll set up the course web site later) up to date with assignments, exercises, etc.
  • Navigation Please try to make things as easy to find as possible
